# Data Mining Project: TMDB Movies Dataset

## Project Overview

This repository contains a **Data Mining project** developed for an academic course. The project is based on the **Full TMDB Movies Dataset**, which includes more than **1.14 million movie records** with structured and unstructured attributes.

The goal of this project is to apply data mining techniques to extract meaningful patterns from large-scale movie data and to build a **content-based movie recommendation system**.

---

## Dataset Information

### Source

Datadruids (2026). *Full TMDB Movies Dataset*. HuggingFace.  
Available at:  
https://huggingface.co/datasets/ada-datadruids/full_tmdb_movies_dataset

---

### Dataset Characteristics

- **1,142,342 movie records**
- **24 attributes**

#### Numerical variables:
- Budget
- Revenue
- Runtime
- Popularity
- Vote average
- Vote count

#### Text-based variables:
- Overview (synopsis)
- Genres
- Keywords

This combination allows both **statistical analysis** and **text mining techniques**.
